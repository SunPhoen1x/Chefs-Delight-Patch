# Polymer Patch for Chef's Delight

This is a patch for [Chef's Delight](https://modrinth.com/mod/chefs-delight), making it work fully server side, with all of its functionality intact.

Chef's Delight is required and not included within this mod.

This mod requires server resource pack, about which setup you can read here: https://polymer.pb4.eu/latest/user/resource-pack-hosting/

You can read more about [Chef's Delight on it's Modrinth page!](https://modrinth.com/mod/chefs-delight)
